================================================================================
                          QUICK START (5 MIN)
================================================================================

1. Install dependencies:
   $ npm install

2. Configure Firebase:
   - Go to Firebase Console
   - Create project & enable Email/Password auth
   - Copy credentials to packages/frontend/.env.local

3. Set up MongoDB:
   - Local: mongod
   - Cloud: MongoDB Atlas (free tier)
   - Copy connection string to packages/backend/.env

4. Start development:
   Terminal 1: cd packages/backend && npm run dev
   Terminal 2: cd packages/frontend && npm run dev

5. Open browser:
   http://localhost:3000
